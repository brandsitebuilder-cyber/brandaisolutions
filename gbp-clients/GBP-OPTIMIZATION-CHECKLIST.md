# GBP Optimization Checklist

## Pre-Setup (Collect from Client)
- [ ] Business name exactly as registered
- [ ] Physical address (must be real, not PO Box)
- [ ] Phone number (must match website exactly)
- [ ] Website URL
- [ ] Business hours
- [ ] 10+ high-quality photos (logo, cover, interior, exterior, team, work examples)
- [ ] List of all services offered
- [ ] Service area suburbs
- [ ] 3-5 unique selling points

## GBP Profile Setup
- [ ] Claim/verify listing on Google Business Profile
- [ ] Business name: EXACT match to real name (no keyword stuffing)
- [ ] Primary category: most specific accurate category
- [ ] Secondary categories: add all that apply (max 9)
- [ ] Address: exact, consistent with website
- [ ] Service area: add all suburbs served
- [ ] Phone: must match website footer exactly
- [ ] Website: full URL with https://
- [ ] Business hours: complete with special hours if applicable
- [ ] Opening date: set if established business

## Description
- [ ] Write 750-character business description
- [ ] Include: what you do, where you serve, what makes you different
- [ ] Include primary keyword naturally
- [ ] NO phone number or URL in description (Google rejects this)
- [ ] NO ALL CAPS or excessive punctuation

## Services
- [ ] Add every service with description
- [ ] Each service has name + brief description
- [ ] Match services listed on website

## Photos
- [ ] Logo: square, clear, recognizable
- [ ] Cover photo: best image, represents brand (1200x900)
- [ ] Interior photos: 3-5 showing workspace/shop (if applicable)
- [ ] Exterior photos: 2-3 showing building/vehicle signage
- [ ] Team photos: 2-3 genuine photos of staff
- [ ] Work photos: 5-10 before/after or project shots
- [ ] All photos: high resolution, well-lit, no stock photos

## Initial Posts
- [ ] Create 3 welcome/intro posts
- [ ] Post 1: "Hello from [Business]!" — introduce the business
- [ ] Post 2: Showcase a recent project with before/after
- [ ] Post 3: Share a practical tip related to the service

## Q&A Section
- [ ] Seed 5-10 common questions with answers
- [ ] Examples: "Do you offer emergency services?", "What areas do you cover?", "How do I get a quote?"

## Products (if applicable)
- [ ] Add products with photos and descriptions
- [ ] Include price range if appropriate

## Verification
- [ ] Google Maps: search business name, verify it appears correctly
- [ ] Check all links work
- [ ] Verify phone number clicks to call
- [ ] Verify directions link works
- [ ] Mobile view: check profile looks good on phone
